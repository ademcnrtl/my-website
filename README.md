# ademcan.dev

- **Framework:** [Next.js](https://nextjs.org/)
- **Deployment:** [Vercel](https://vercel.com)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)

## Overview

- `app/*` - All other pages.
- `components/*` - Components.
- `public/*` - The folder I use for photos.
- `lib/*` - Includes data and supporting tasks.

## Demo
 
```bash
https://ademcan.dev
```

## Running Locally

```bash
git clone https://github.com/ademcnrtl/my-website.git
cd my-website
npm install
npm run dev
```
Create a `.env` file similar to [`.env.example`](https://github.com/ademcnrtl/my-website/blob/master/.env.example).

## Getting Started

First, run the development server: 

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

To see the app in action, head over to http://localhost:3000 in your browser.

You can kick off your development by tweaking app/page.tsx. The page will automatically live-reload as you make changes.

Check out our [Next.js deployment documentation](https://nextjs.org/docs/deployment) for more details.
